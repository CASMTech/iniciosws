HyperText Markup Language
«Lenguaje de marcado de hipertexto», hace referencia al lenguaje de marcado para la elaboración de páginas web.

Cascading Style Sheets
«Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito HTML

JavaScript
